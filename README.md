# password-tk

# 安全密码生成器（Password Generator）

一款基于 Python + Tkinter 开发的可视化密码生成工具，支持自定义密码长度、灵活选择是否包含特殊符号，生成的密码自动复制到剪贴板，兼顾安全性与易用性。

## 功能特点
- **安全可靠**：强制密码包含大小写字母 + 数字，可选特殊符号，避免弱密码
- **灵活配置**：支持 4-32 位密码长度设置（超出范围自动调整为合理值）
- **便捷高效**：生成密码后自动复制到剪贴板，无需手动复制
- **可视化界面**：简洁直观的 GUI 操作，无需命令行基础，小白也能轻松使用
- **容错性强**：输入无效长度时自动使用默认12位密码，避免程序报错

## 环境依赖
- Python 3.6 及以上版本
- 依赖库：`pyperclip`（用于剪贴板功能）

## 安装步骤
1. 克隆本仓库到本地
   ```bash
   git clone https://github.com/FOREVERbh/password-tk.git
   cd password-tk
   ```

2. 安装依赖库
   ```bash
   pip install pyperclip
   ```

3. 运行程序
   ```bash
   python password.py
   ```

## 使用说明
1. 运行程序后，将弹出密码生成器窗口
2. （可选）修改密码长度（默认12位，支持4-32位）
3. （可选）勾选/取消「包含特殊符号」（默认勾选）
4. 点击「生成密码」按钮，即可生成密码
5. 生成成功后，密码会显示在窗口中，且已自动复制到剪贴板，直接粘贴使用即可

## 注意事项
- 若运行时提示 `pyperclip` 相关错误，需确保已正确安装依赖库
- 密码长度建议至少8位，结合大小写字母、数字和特殊符号，安全性更高
- 特殊符号包含 `!@#$%^&*()_+-=[]{}|;:,.<>?` 等常见符号，适配多数平台密码规则
